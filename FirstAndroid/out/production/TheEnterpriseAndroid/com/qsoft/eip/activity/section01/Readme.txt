Section 01 - GUI Mapping

Overview:
    This section introduce the basics setup for mapping activities and fragments with its layout.
Guideline:
    SimpleActivity01 - This is the simplest mapping layout, with only one @ViewMapping annotation
    SimpleActivity02 - This one include one field mapping with a component in layout
    SingleFragmentActivity - This one is used for storing a single fragment within, intend to use with
        fragments in this section tutorials.
    SimpleFragment01 - This one demonstrate the usage of GUI Mapping with fragment component. It's as simple as
        SimpleActivity01
    SimpleFragment02 - This one is the same as SimpleActivity02
